# DLithe 14-Day Placement Training

This repository contains C++ source files, coding challenge solutions, and programming exercises completed during the **DLithe 14-Day Placement Training** program.

## 📂 Repository Structure

Below is an overview of the daily progress and topics covered:

### [Day 2](file:///Day%202-%2030.05/) - Object-Oriented Programming (OOPs) & Patterns
Focus on basic constructors, constructor overloading, and nested loop pattern exercises.
* **OOPs & Constructors**:
  * [constructortest.cpp](file:///Day%202-%2030.05/constructortest.cpp) — Introduction to constructors
  * [overloadconstruct.cpp](file:///Day%202-%2030.05/overloadconstruct.cpp) — Constructor overloading
  * [rectconstructor.cpp](file:///Day%202-%2030.05/rectconstructor.cpp) — Rectangle class initialization
* **Pattern Programs**:
  * [menu.cpp](file:///Day%202-%2030.05/menu.cpp) — Interactive menu-driven program
  * [squarepattern.cpp](file:///Day%202-%2030.05/squarepattern.cpp) — Square shape star pattern
  * [v_pattern.cpp](file:///Day%202-%2030.05/v_pattern.cpp) — Alphabetical/star V pattern
  * [vivekpattern.cpp](file:///Day%202-%2030.05/vivekpattern.cpp) — Custom shape pattern

## 🛠️ How to Compile and Run
Ensure you have a C++ compiler (like `g++`) installed on your system. 

Navigate to the day's directory and run:
```bash
g++ <filename>.cpp -o output
./output
```
